# Week 1 - React Hands-on

## Exercise

Create Home, About, and Contact components using React Class Components.

## Objective

To understand React Class Components, component creation, rendering JSX, and component integration in a React application.

## Technologies Used

* React
* JavaScript
* HTML
* CSS
