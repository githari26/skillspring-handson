# Exercise 1 - Configuring Basic Spring Application

## Objective
Learn the fundamentals of the Spring Framework by configuring beans using XML-based configuration.

## Concepts Covered
- Spring Core
- Inversion of Control (IoC)
- Dependency Injection (DI)
- XML Bean Configuration
- Maven Project Structure

## Components
- BookRepository
- BookService
- MainApp

## Output
The application loads Spring beans from `applicationContext.xml` and demonstrates successful bean creation and dependency injection.

## Technologies Used
- Java
- Spring Framework
- Maven
- IntelliJ IDEA
